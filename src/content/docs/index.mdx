This platform is build for aggregatation of free rescoures across various stearms and multiple professions, 
Make them accesible to everyone. This platform is made open-source so as to support valuable contibutions for further growth.

## Contribution

If you are familiar with github go ahead and follow the steps below, else fill out this [form](https://forms.gle/W59k42MmdfGuLbSZ6).

### Github
---
    <Steps>
        1. Go ahead and fork this [repo](https://github.com/NarayanaRavada/E-Lib)

        2. Clone the forked repo to you local machine.

            ```bash
            git clone your-forked-repo-link
            ```

        3. Add or change the resources that you would like to contribute. changes should be made in `src/content/docs`
           in respective category or create a new folder if category does not exist.

        4. push the changes to forked repo and make a pull request, we verify it and merge.
    </Steps>
